# Screenshots needed

Take these before posting publicly. Use macOS `⇧⌘4` (area screenshot) or `⇧⌘4 Space` (window screenshot with shadow).

Aim for a 2x / Retina display so they look sharp on GitHub.

---

## menu.png
The main menu open, showing:
- Several models listed (mix of MLX and GGUF)
- One model active (checkmark or green dot)
- The menu bar title showing live stats: `⚡ 42 t/s 14%ctx`
- Submenus visible if possible (Select / Hide / Settings… / Benchmark…)

**Tip:** Use a window screenshot (`Space` after `⇧⌘4`) on the open menu.

---

## quicktest.png
The Quick Test Prompt window mid-stream or just after a response, showing:
- A prompt in the input field
- A streamed response in the output area
- TTFT / tok/s / context stats visible at the bottom

Bonus: capture the benchmark results chart (Chart.js bar chart) as a third screenshot `benchmark.png` and add it to the README.
